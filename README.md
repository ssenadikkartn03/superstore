Superstore Dashboard is an interactive data analysis application built with Python, designed to explore and visualize sales and profit metrics from the popular Sample - Superstore dataset. This project aims to provide actionable insights through clean visualizations and intuitive layout, making it ideal for data analysts, business intelligence professionals, and decision-makers.
Using tools like pandas, matplotlib, seaborn, and Plotly Dash, the dashboard enables users to:
- Examine profit distribution across orders
- Compare performance across product categories and sub-categories
- Analyze regional profitability
- Identify loss-making segments and potential areas for improvement
